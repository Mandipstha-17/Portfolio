## 📂 Project Setup

To run this portfolio locally, follow these steps:

```bash
# Clone the repository
git clone https://github.com/Mandipstha-17/portfolio.git

# Change directory
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev

```
